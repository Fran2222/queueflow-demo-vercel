# Sanitization Report

This Vercel-ready demo was rebuilt as a new frontend-only project.

## Removed / Not Included

- Original client names and branding
- Original logos, banners, and client assets
- SQLite database files
- Backend Python/Flask application code
- Printer integrations and local printer scripts
- Windows batch/PowerShell startup files
- Installers and offline packages
- Real user accounts and passwords
- Local machine paths
- Debug logs and cache files
- Any `.env` secrets

## Included

- Generic queue management UI
- Mock service areas
- Mock queue data
- React/Vite frontend
- CSS animations and responsive layout
- localStorage-based demo state
- Vercel SPA routing config

## Safe for Public GitHub?

Yes, this folder is designed to be safe for public GitHub as a frontend demo. Review the repository before pushing if you add new files later.
